# Weather Stone (날씨돌)

> "밖에 나가지 않아도 돌만 보면 날씨를 안다"

## 컨셉

실제 돌멩이 하나가 화면에 놓여 있다. 이 돌은 현재 날씨를 온몸으로 표현한다.
비가 오면 젖어 있고, 해가 나면 말라 있고, 눈이 오면 눈이 쌓여 있다.
텍스트나 아이콘이 아닌, **돌의 물리적 상태 변화**만으로 날씨를 전달하는 앱.

---

## 핵심 기능

### 1. 실사 3D 돌 렌더링
- 포토리얼리스틱 질감의 돌 하나를 화면 중앙에 배치
- PBR(Physically Based Rendering) 머티리얼 적용
- 사용자가 드래그로 돌을 회전하며 관찰 가능

### 2. 날씨 연동 상태 변화
현재 위치의 실시간 날씨에 따라 돌의 외형이 변한다.

| 날씨 | 돌의 상태 |
|------|----------|
| 맑음 | 건조하고 밝은 돌, 따뜻한 조명 |
| 흐림 | 약간 어두운 톤, 부드러운 그림자 |
| 비 | 표면이 젖어 반사광 증가, 물방울 이펙트 |
| 눈 | 윗면에 눈 적층, 서리 질감 |
| 바람 | 돌 주변 먼지/낙엽 파티클 |
| 안개 | 돌 주변 안개 이펙트, 낮은 가시성 |
| 폭염 | 돌 표면 아지랑이, 갈라진 질감 |
| 한파 | 돌 표면 결빙, 푸른 톤 |

### 3. 위치 기반 날씨 데이터
- 사용자 위치 자동 감지 (GPS)
- 수동 위치 검색 지원
- 실시간 날씨 API 연동

### 4. 부가 정보 (최소한으로)
- 돌을 탭하면 온도/체감온도 표시
- 화면 하단에 간결한 텍스트: "비, 18°C"
- 평소에는 돌만 보이는 미니멀 UI

## 기술 스택 (후보)

| 영역 | 후보 기술 |
|------|----------|
| 플랫폼 | iOS (Swift) / 크로스플랫폼 (React Native, Flutter) |
| 3D 엔진 | Three.js (웹), SceneKit (iOS), Unity |
| 날씨 API | OpenWeatherMap / WeatherAPI / Apple WeatherKit |
| 3D 모델링 | Blender로 돌 모델 + 텍스처 제작 |

> 플랫폼과 기술 스택은 추후 확정

## 화면 구성

```
┌─────────────────────────┐
│                         │
│                         │
│                         │
│        🪨 [3D 돌]       │
│                         │
│                         │
│                         │
│                         │
├─────────────────────────┤
│   비, 18°C  서울특별시    │  ← 탭 시 표시/숨김
└─────────────────────────┘
```

- 기본 상태: 돌만 풀스크린으로 표시
- 배경: 날씨에 맞는 조명/색감 (하늘 아님, 단색 또는 그라디언트)
- 인터랙션: 드래그로 회전, 탭으로 정보 토글

---

## 사용자 시나리오

1. **아침에 앱을 연다** → 돌이 젖어 있다 → "아, 비 오는구나" → 우산 챙김
2. **겨울 아침** → 돌에 서리가 끼어 있다 → "많이 춥겠다" → 두꺼운 옷 착용
3. **한여름** → 돌이 갈라지고 아지랑이가 피어오른다 → "폭염이구나"

---

## 차별점

- **정보 과잉의 반대**: 숫자, 그래프, 예보 없이 직관적 날씨 인지
- **감성적 경험**: 기능이 아닌 오브제를 보는 느낌
- **한 화면, 한 오브젝트**: 앱을 열자마자 0.5초 안에 날씨 파악

---

## 향후 확장 가능성 (미확정)

- 돌 커스터마이징 (화강암, 대리석, 자갈 등)
- 위젯 지원 (홈 화면에 작은 돌)
- Apple Watch 컴플리케이션
- 시간대별 변화 (아침 이슬 → 낮 건조)
- 사운드 (빗소리, 바람소리 등 ASMR)
